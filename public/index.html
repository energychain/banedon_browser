Getting Started Guide

1. Install Browser Extension

Download the extension package and install it in your browser:

1. Download extension from the button above 2. Extract the ZIP file 3. Open Chrome ‚Üí Extensions (chrome://extensions/) 4. Enable "Developer mode" 5. Click "Load unpacked" and select the extracted folder

2. Create a Session

First, create a session to get a session ID:

POST /api/sessions Content-Type: application/json { "metadata": { "browser": "chrome", "purpose": "testing" } }

3. Configure Extension

Enter the session ID in the browser extension popup and connect.

4. Send Commands

Use the session ID to send automation commands:

POST /api/sessions/{sessionId}/commands Content-Type: application/json { "action": "navigate", "url": "https://example.com" }
