üéØ Best Practices

Writing Effective Tasks

  • Be specific: "Click the red 'Subscribe' button" vs "Click subscribe"
  • Use complete URLs: "Go to https://example.com" vs "Go to example"
  • Break complex tasks: Split multi-step processes into individual tasks
  • Include context: "In the search results, click the first Amazon listing"

Extension vs Server Mode

üîå Extension Mode (Recommended): Tasks run in your actual browser. You can see what's happening in real-time and maintain login sessions.
ü§ñ Server Mode: Tasks run in a server-side browser. Useful for headless automation but can't access your logged-in sessions.

Session Management

  • Reuse sessions: Enter the same session ID to continue where you left off
  • New sessions: Leave session ID empty to create a fresh session
  • Extension sessions: Connect your extension first, then use that session ID

üîß Troubleshooting

Extension Not Connecting

  1. Reload the extension in chrome://extensions/
  2. Check that the server URL is correct: wss://browserless.corrently.cloud/ws
  3. Try creating a new session ID
  4. Check the browser console for WebSocket errors

Tasks Not Executing

  • Ensure the extension shows "Connected" status
  • Check that the session ID matches between extension and interface
  • Make sure the website is accessible and loaded
  • Try simpler tasks first to test the connection

Screenshots Not Showing

  • The working tab must be visible for screenshots
  • Reload the extension if recently updated
  • Check browser permissions for the extension
